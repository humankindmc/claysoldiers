Clay Soldiers Port Notes
========================

This plugin is starting as a Paper server-side approximation of the Clay Soldiers Mod:

- Source repository: https://github.com/SanAndreaP/ClaySoldiersMod
- Original mod authors listed in `mcmod.info`: SanAndreasP, SilverChiren, CliffracerX
- Original credits listed in `mcmod.info`: KodaichiZero
- License template found in the repository: Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International

Current implementation status:

- No original raster assets are copied into this repository yet.
- The first slice ports gameplay concepts and balancing constants using normal Paper API entities/items.
- If original textures/models are copied later for a resource pack, keep attribution and non-commercial/share-alike requirements with the distributed pack.
